IndexOS Documentation

Welcome to the comprehensive documentation for IndexOS, the easy-to-use Linux distribution based on Arch Linux. This guide will help you get started with IndexOS, from installation to advanced configuration.

New to Linux? IndexOS is designed to be beginner-friendly while maintaining the power of Arch Linux. Don't worry if you're new to Linux - we've got you covered!

System Requirements

Before installing IndexOS, make sure your system meets the minimum requirements:

Component Minimum Recommended
CPU x86_64 processor, 1 GHz Multi-core, 2+ GHz
RAM 2 GB 4+ GB
Storage 20 GB 50+ GB SSD
Graphics Basic VGA Dedicated GPU
Network Ethernet or Wi-Fi Ethernet connection
UEFI vs BIOS: IndexOS supports both UEFI and legacy BIOS systems. UEFI is recommended for newer hardware.

Download & Preparation

1. Download IndexOS ISO

Download the latest IndexOS ISO from our official website:

📥 Download IndexOS 2025.1 📋 Verify Checksum

2. Create Bootable USB

You'll need a USB drive with at least 4GB of space. Use one of these tools:

  • Rufus (Windows): Easy-to-use tool for creating bootable USB drives
  • balenaEtcher (Cross-platform): Simple and reliable
  • dd command (Linux/macOS): Command-line tool for advanced users
bash
# Using dd command (Linux/macOS)
sudo dd if=indexos-2025.1.iso of=/dev/sdX bs=4M status=progress

# Replace /dev/sdX with your USB device
# Use 'lsblk' to identify your USB device
Warning: Double-check your USB device path! The dd command will overwrite all data on the specified device.

Installation Methods

IndexOS offers multiple installation methods to suit different user preferences:

Graphical Installer (Recommended)

The easiest way to install IndexOS with a user-friendly graphical interface:

  1. Boot from USB drive
  2. Select "Install IndexOS" from the boot menu
  3. Follow the on-screen instructions
  4. Reboot and enjoy!

CLI Installer (Advanced)

For users who prefer command-line installation:

bash
# Boot into live environment and run:
sudo indexos-install

# Follow the interactive prompts

Manual Installation

Complete control over the installation process (for experts):

  • Partition disks manually
  • Configure filesystems
  • Install base system
  • Configure bootloader

Boot from USB

Accessing Boot Menu

To boot from your USB drive, you'll need to access your computer's boot menu:

Manufacturer Boot Menu Key BIOS/UEFI Key
ASUS F8, Esc F2, Delete
Dell F12 F2
HP F9, Esc F10
Lenovo F12 F1, F2
MSI F11 Delete
Tip: Press the boot menu key repeatedly while your computer starts up. The timing can be tricky!

Package Management

IndexOS uses pacman as its package manager, with access to both official Arch repositories and the AUR.

Basic Package Operations

bash
# Update system
sudo pacman -Syu

# Install a package
sudo pacman -S package-name

# Remove a package
sudo pacman -R package-name

# Search for packages
pacman -Ss search-term

# List installed packages
pacman -Q

AUR Helper - yay

IndexOS comes with yay pre-installed for easy AUR access:

bash
# Install AUR package
yay -S aur-package-name

# Update system including AUR packages
yay -Syu

# Search AUR
yay -Ss search-term
Pro Tip: Always update your system regularly with sudo pacman -Syu to keep your system secure and up-to-date.

Customization

Easily customize your desktop through System Settings:

  1. Right-click on desktop → Configure Desktop and Wallpaper
  2. Open System Settings for comprehensive options
  3. Install themes from KDE Store
  4. Customize panels and widgets
bash
# Install additional desktop environments
sudo pacman -S gnome          # GNOME Desktop
sudo pacman -S xfce4          # XFCE Desktop
sudo pacman -S i3-wm          # i3 Window Manager

Troubleshooting

Common Issues

Boot Issues

Problem: System won't boot after installation

bash
# Boot from live USB and mount your system
sudo mount /dev/sdaX /mnt
sudo arch-chroot /mnt

# Reinstall bootloader
grub-install /dev/sda
grub-mkconfig -o /boot/grub/grub.cfg

Network Issues

Problem: No internet connection

bash
# Check network interface
ip link show

# For Wi-Fi
sudo systemctl enable NetworkManager
sudo systemctl start NetworkManager
nmcli device wifi connect "SSID" password "password"

# For Ethernet
sudo systemctl enable dhcpcd
sudo systemctl start dhcpcd

Audio Issues

Problem: No sound output

bash
# Check audio devices
aplay -l

# Install PulseAudio (if not installed)
sudo pacman -S pulseaudio pulseaudio-alsa

# Restart audio service
pulseaudio -k
pulseaudio --start
